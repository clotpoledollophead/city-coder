🏙️ 如何使用 CodeScape

介面使用說明 · Interface Guide
⌨️ 左側：程式碼編輯區
▶ 執行 執行程式碼，在右側地圖建造建築（也可按 Ctrl+Enter）
✎ 清除程式碼 清空左側編輯區的程式碼（地圖不受影響）
🗑 清除地圖 移除地圖上所有建築（程式碼不受影響）
📚 函式庫 點擊卡片可將函式插入程式碼；鎖定的卡片需完成課程解鎖
🗺️ 右側：3D 地圖操作
🖱 左鍵拖曳 平移地圖視角
🖱 右鍵拖曳 旋轉地圖視角
🖱 滾輪 縮放地圖
⌂ Reset 重置視角到預設位置
懸停 滑鼠移到格子上會顯示該格的 row, col 座標
⚠️ 驚嘆號 蓋了一些建築後，部分建築上方會出現閃爍驚嘆號，點擊開始 Debug 挑戰！
💡 座標規則：只能在島嶼陸地上建造（含沙灘）。懸停地圖格子可查看座標， 有效範圍約 row 25–55, col 25–55，但確切邊界依地形而定。 若座標在海洋中，執行時會顯示錯誤提示。

🐍 支援完整 Python 語法：可使用 for、while、 if/elif/else、def、變數、串列、字典，以及 print()！
🖱 左鍵拖曳：平移 🖱 右鍵拖曳：旋轉 🖱 滾輪：縮放
