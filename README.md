# city-coder

A browser-based city builder designed to teach kids coding through play. Inspired by SimCity, users script their own miniature world with Python-like commands which are translated into Three.js building operations behind the scenes.

This project acts as a **bridge between block-based coding and text-only scripting**: learners can begin with visual block environments (code generation not shown here) and then move into writing text scripts directly while still enjoying immediate visual feedback.

### What we hope to achieve

1. A fun, interactive environment where building a city reinforces programming logic.
2. A gentle transition path from drag‑and‑drop blocks to writing code by hand.
3. Motivation for students to explore more advanced scripting and game development.
